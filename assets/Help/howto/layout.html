Layout

The first of two windows that you will see open is the console window (figure 2.1). This window is need by the Virtual Breadboard to work. This window should not be close until you have finished using the program or are instructed to do so, however you may minimize it.


Figure 2.1: The console window is the first window you will see

The second window (Figure 2.2) is the actual window you will be working with. Inside it there are two more windows intitled "Tool Box" (Figure 2.3) and "Breadboard" (Figure 2.4). In short the tool box contains the components that may me used on the breadboard.


Figure 2.2: The Virtual Breadboard window is the second window you should see
Figure 2.3: Tool box contains the components that may be used on you Virtual Breadboard Figure 2.4: The breadboard is the component where all your circuitry is tested

Take note in Figure 2.3 that the currently selected tool has a black box around it.
