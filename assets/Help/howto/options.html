Using

Some tools have options that can be change. See table 4.1 for a list

Tool Options
LED You may select the color of the LED to be used.
Resistors The amount of resistance to be supplied.
Wires You may select the color of the wire to be used.
Tester No customizable options but it displays infomation.

Some options just provide you with more options on how a tool will look (for example wires) and others will control how it acts (for example resistors).
