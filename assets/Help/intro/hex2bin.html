Hexidecmial to Binary

To convert a Hexidecimal (base 16) number to Binary (base 2). Each alfa-numaric charater represents one nibble. Each nibble is converted to decimal using the below chart. Then the decimal numbers are converted into binary.

Example 1: Convert 1B to binary
Hex Number: 1B
Nibble # Hexidecimal # Decimal # Binary #
1 1 1 0001
2 B 11 1011
Binary Number: 0001 1011

Example 2: Convert 38 to binary
Hex Number: 38
Nibble # Hexidecimal # Decimal # Binary #
1 3 3 0011
2 8 8 1000
Binary Number: 0011 1000
