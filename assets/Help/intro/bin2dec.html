Binary to Decimal

To convert a Binary (base 2) number to Decimal (base 10) number you can use the following method. Create a chart as below. There should be as many colums as 1s and 0s in the number. Lable the colums from right to left in the following order 1, 2, 4, 8, 16, 32, 64, 128. If the numbers exceed 8 bits use the following formula to continue it v=n2. Where v is the value and n is the colum number it is for. When you write the binary number in the chart where ever there is a 1 add together all the colums values. The sum of the numbers with a 1 is the decimal number.

128 64 32 16 8 4 2 1









Example 1: Convert 11011 to Decimal
16 8 4 2 1
1 1 0 1 1
16 + 8 + 2 + 1 = 27
Decimal Number: 27

Example 2: Convert 111000 to decimal
32 16 8 4 2 1
1 1 1 0 0 0
32 + 16 + 8 = 56
Decimal Number: 56
Note: When you get better you will not need the chart!
