Decimal to Binary

To convert a Decimal (base 10) number to binary (base 2) you can use a method called successive division. To use this method you will take your decimal number and divide it by the number 2. If the number is a whole number then it equals a 0, if it is not a whole number then it equals a 1. Continue this till the number is no longer divisible by 2.

Example 1: Convert 27 to binary
Number Math Binary reptesentation
27 27 / 2 = 13.5 1 Last Number
13 13 / 2 = 6.5 1
6 6 / 2 = 3 0
3 3 / 2 = 1.5 1
1 1 / 2 = 0.5 1 First Number
Binary Number: 11011

Example 2: Convert 56 to binary
Number Math Binary reptesentation
56 56 / 2 = 28 0 Last Number
28 28 / 2 = 14 0
14 14 / 2 = 7 0
7 7 / 2 = 3.5 1
3 3 / 2 = 1.5 1
1 1 / 2 = 0.5 1 First Number
Binary Number: 111000
Note: In most cases you will be required to answer a question in 8 bits. In neither example are there 8 bits. To fix this we add "Leading Zeros". Leading zeros are zeros in front of the number to make it 8 bits long. For example 11011 would become 0001 1011 and 111000 would become 0011 1000. Use of these leading zeros do not change the value of the number.
