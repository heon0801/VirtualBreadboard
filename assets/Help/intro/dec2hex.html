Decimal to Hexidecmial

To convert a Decimal (base 10) number to hexidecmial (base 16) first convert the number to binary (see Decimal to Binary). Convert the number to the make the length of the binary number a multipul of 4 (or round up to the nearest nibble). Once that is done convert each nibble to decimal (See Binary to Decimal). Then use the below chart to come up with the correct value. Any number between 0 and 9 will stay the same but numbers from 10 to 15 will have a letter to represent them.

Example 1: Convert 27 to hex
Number: 27
Binary reptesentation: 0001 1011
Nibble # Binary # Decimal # Hexidecimal #
1 0001 1 1
2 1011 11 B
Hex Number: 1B

Example 2: Convert 56 to hex
Number: 56
Binary reptesentation: 0011 1000
Nibble # Binary # Decimal # Hexidecimal #
1 0011 3 3
2 1000 8 8
